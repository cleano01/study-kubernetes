# study-kubernetes
---
Repository of registry about study of Kubernetes.

## Requirements

- Docker version:

  > DOCKER >=  20.10.23

- Minikube version:

  > MINIKUBE >= 1.28.0
  
- Kubectl version:

  > KUBECTL >= 4.5.7


## Execution Minikube

- Execution command bellow:

  > minikube start --driver=docker
